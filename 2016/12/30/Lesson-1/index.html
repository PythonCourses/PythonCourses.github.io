Functions

Functions are simply blocks of code that we can execute anywhere in our program. It allows us to break our program up in to tasks, so when we want to accomplish the task, we can simply call the function.

1-6: A Simple Function

Lets start of with a basic function. We create functions using the def keyword, followed by the function name, then parenthesis, then a colon. We then indent. When we indent, we are putting the indenting code into a block . In this case, everything in the this block, which is just the print command, is part of the function.
