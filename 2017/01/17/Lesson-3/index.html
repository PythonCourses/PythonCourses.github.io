Lesson-3

This lesson will cover:

Modulus Operator

3-1 What the modulus operator does

The modulus operator is a little weird, very simple, and suprisingly useful. When used between two Ints, it returns the remainder from the division of the left Int over the right Int. Lets try a basic modulus operation.

Because 13 goes in to 5 twice, with a remainder of 3, we see that 13 % 5 is 3.

3-2 Determining if an Int is odd or even

An even Int is a Int when divided by 2, has no remainder. While an odd Int will have a remainder of 1. So we can use the modulus operator and en if/else statment, to determine if a Int is even or odd. Lets allow our program to take a Int from the user, then tell the user if the Int is even or odd

While Loops

3-3 How While Loops work

While loops are loops that execute while a given condition is true. They are useful if we want to repeteadley execute a block of code, until a value has changed. Lets take a look at a simple while loop

The num starts at 0, and each time it goes through the loop, it increases by 1. It does this while num is less than 10. So once it hits ten, the loop stops.

Functions

Functions are simply blocks of code that we can execute anywhere in our program. It allows us to break our program up in to tasks, so when we want to accomplish the task, we can simply call the function.

When we indent, we are putting the indenting code into a block . In this case, everything in the this block, which is just the print command, is part of the function.
