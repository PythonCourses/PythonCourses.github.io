Lesson-4

This lesson will cover:

  1. Object oriented programming using classes and **objects
  2. Defining and implementing attributes for a class
  3. Defining and implementing methods for a class

Our First Class

4-1 Structure of a Class

A class is defined using the class keyword, then followed by the class name that starts with a capital leter. The attributes of a class are defined in the init function. So let’s create a basic class called Smith, a person who has the following attributes:

  • lastName
  • age
  • firstName

4-2 Defining a Class Method

So far, our class Smith has all of the attributes we need, but we also want to be able to do something with this data. This is where methods come in handy. Methods are functions that are specific to a class, and therefore have access to the attributes of that class. So, lets give the Smith class the ability to print out its name, by defining a method sayName. Also, because the firstName attribute is initialized empty, lets create a method to change the firstName attribute, called setName(), that takse in a name paramater

4-3 Creating an object from a Class

Classes are frameworks to create objects from. Essentially, we can create as many Smith objects that we want, which all have the lastName, age, and firstName attributes and the setFirstName() and sayName() methods. So lets create our first Smith object using the initializer.Then change its firstName using the setFirstName() method. And finally, say its name using the sayName() method.

4-4 More methods…

Right now, our Smith objects never get older than 0 years old, they are infinitely babies…Lets change that! We are going to create a method called haveBirthday() that increases the age by 1. Also, we need a sayAge() function, so we can see how old the Smith object is. We are going to call the haveBirthdayMethod() 18 times using a for loop.

4-5 Multiple Objects

One of the key characteristics of classes, is the ability to create multiple objects from the class. So, lets try creating multiple Smith objects (each with a different first name and number of birthdays).
