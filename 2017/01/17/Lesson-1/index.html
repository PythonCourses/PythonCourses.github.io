Lesson 1

This lesson is all about the basics. We are going to accomplish alot! We are going to be:

  1. Printing so we can see the output of our program
  2. Assigning number values to variables using Ints
  3. Assigning text values to variables using Strings
  4. Performing mathematical operations on variables
  5. Changing variable types using Typecasting
  6. Creating and accessing elements in a List
  7. Adding and deleting items from the list

This may seem like a lot to handle, but you will pick up on it pretty quickly. So lets get started.
