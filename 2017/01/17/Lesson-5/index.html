Lesson 5

This lesson is all about using python to draw shapes onto a canvas. A canvas is simply a highly customizable, empty screen that we can draw shapes on. And eventually, we will have these shapes move across the screen to create our game!

We are going to be:

  1. Crating a canvas on our computer screen
  2. Using Canvas methods to create a variety of shapes
  3. Understanding how numerical values represent positions on the canvas

Drawing with Python

5-1 Bringing up the Canvas

The first step to drawing in Python is creating something to draw on, this is the canvas. We can create a canvas fromt the Tkinter module. First, we create a tk object, which gives us access to all of the Tkinter methods. Then, we create a canvas by giving the Canvas() method our tk object, a width, and a height.

Running this program should bring up an empty canvas that is 600 pixels wide and 400 pixels tall.

How Pixels are represented with Points

Take a look at the image above. In order to capture positions on the canvas, we must understand how the position is represented using points. A Point is a combination of an x-value and a y-value. The x-value represents the horizonal point (how far right the point it is), and the y-value represents the vertical point (how far down the point is). So, the top left corner has an x-value of 0 and a y-value of 0, so the point is (0,0). Moving right from this corner will increase the x-value, and moving down from this corner will increase the y-value. The highest possible x-value is 600, because the canvas is 600 pixels wide, and the highest y-value is 400, because the canvas is 400 pixals tall.

5-2 Drawing our first shape.

The simplest shape we can start with is a line. we can create a line by givving the create_line() method two paramaters, the first point and the second point. The method will then draw a line from the first point to the second point. We need to call the create_line() method after we create our canvas, but before we call canvas.pack(). Lets make the first point (10,10) and the second point (50,100). So the line will be in the top left of our canvas.

5-3 Drawing a rectangle

Lets make another shape. The next simplest shape we can draw is a rectangle. Just like the create_line() method, we need to give the create_rectangle()two points; the top left point and the top right point. Also, we can optionally give the create_rectangle() method a “fill” paramater, which takes in a color, to fill the rectangle with.
