Lesson 9

In this lesson, we will be adding a scoring system to our Paddle Ball game. Right now, our Paddle Ball game has the ball bouncing off the paddle and the walls, and we have control over the paddle. However, this game would be much more fun if it had a scoring system, so we could go for a high score!

9-3 Actually showing the score

The score label is in the right place, but it does not show any number….. kinda useless. To create a score, we need to create an Int variable called score that will increase by one every time the ball bounces off the paddle. Then, we need to ensure the score label shows the score variable by making the text of the score label be ** “Score :” + str(score). Also, when we increase the score variable, we need to update score label variable as well. This label update is accomplished in an unusual way; we will delete the label and recreate it in the same location, but with the new score.
