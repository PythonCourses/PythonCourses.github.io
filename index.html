Hi students, welcome to the lesson page for Intro to Python at Code for Fun!

These lessons are intended to be used as a follow-up after each week’s in-class lesson. So Lesson 1 covers the same topics as the 1st week’s class, Lesson 2 covers the same topics of the 2nd week’s class, and so on….

  4. Once installed, locate the IDLE application on your computer
  5. Open IDLE, check to make sure that the version number is 3.4 or higher.
  6. Click File -> New file to open a new file. **

Follow the lesson and make a new file for each section, continue naming the files Lesson1-1, Lesson1-2, and so on.

Good luck and see you all in class!

Lesson-7

In this lesson, we will create a controllable rectangle called a Paddle, that will move from side to side, and is controllable using the arrow keys. This Paddle will function similar to the paddle in brick breaker.

We are going to be:

  1. Crating a canvas on our computer screen
  2. Creating a “Paddle” class to provide attributes and functionality to the Paddle.
  3. Using Canvas methods to create the Paddle’s shape
  4. Use key-binding to attach key press to the Paddle’s method

It’s Paddle Time

7-1 Bringing up the Canvas

Before we create our Paddle class, let’s make sure we can bring up the canvas, so that we can draw our Paddle on to it.

Running this program should bring up an empty canvas that is 1000 pixels wide and 800 pixels tall.

7-2 Creating the Paddle Class

Creating a class for our Paddle will give us control over its attriubutes (like its size and speed), and its methods (like its ability to move left and right). So lets start by creating an init function that creates an canvasm idm abd xSpeed attribute. The canvas attribute will tell the the Paddle which canvas to appear on (which will be the empty canvas we made). The id attribute will be the rectangle shape itself. And the xSpeed will be the speed of the paddle (…once we get it moving). This code should simply make the paddle appear on the screen.

Note: Pay attention to the coordinates that we give the create_rectangle() function. These coordinates puts the rectangle in the bottom, middle of the screen.

7-3 Moving the Paddle

To move the paddle, we need to do a few things. First we need to import time at the top of the file, so we can create an animation loop. Next, we must create a draw() method in the Paddle class, that draws moves the Paddle by xSpeed pixels. this draw() method will also check the far left and far right coordinates of the paddle, and if the far left coordinate is less than 0, we know the paddle has hit the left side of the canvas, so we must reverse the direction to make it bounce. If the far right coordinate of the paddle is greater than the canvis width, than we must also reverse the direction here.
Finally, we must create the animation loop which draws the paddle, updates the canvas, and sleeps the program for .01 seconds.

7-4 Controlling the Paddle

It’s cool that the Paddle is moving, but it would be a whole lot cooler if we could control the Paddle someway. Luckily, there is an easy way to do this; it’s called key-binding. Key binding simply assigns a function or method to a key on the keyboard. In this case, we will tell the left key to change the Paddle’s xSpeed attribute to -3, and the right key will change the Paddle’s xSpeed attribute to 3. We will accomplish this binding in the Paddle’s init method.

Lesson 5

This lesson is all about using python to draw shapes onto a canvas. A canvas is simply a highly customizable, empty screen that we can draw shapes on. And eventually, we will have these shapes move across the screen to create our game!

We are going to be:

  1. Crating a canvas on our computer screen
  2. Using Canvas methods to create a variety of shapes
  3. Understanding how numerical values represent positions on the canvas

Drawing with Python

5-1 Bringing up the Canvas

The first step to drawing in Python is creating something to draw on, this is the canvas. We can create a canvas fromt the Tkinter module. First, we create a tk object, which gives us access to all of the Tkinter methods. Then, we create a canvas by giving the Canvas() method our tk object, a width, and a height.

Running this program should bring up an empty canvas that is 600 pixels wide and 400 pixels tall.

How Pixels are represented with Points

Take a look at the image above. In order to capture positions on the canvas, we must understand how the position is represented using points. A Point is a combination of an x-value and a y-value. The x-value represents the horizonal point (how far right the point it is), and the y-value represents the vertical point (how far down the point is). So, the top left corner has an x-value of 0 and a y-value of 0, so the point is (0,0). Moving right from this corner will increase the x-value, and moving down from this corner will increase the y-value. The highest possible x-value is 600, because the canvas is 600 pixels wide, and the highest y-value is 400, because the canvas is 400 pixals tall.

5-2 Drawing our first shape.

The simplest shape we can start with is a line. we can create a line by givving the create_line() method two paramaters, the first point and the second point. The method will then draw a line from the first point to the second point. We need to call the create_line() method after we create our canvas, but before we call canvas.pack(). Lets make the first point (10,10) and the second point (50,100). So the line will be in the top left of our canvas.

5-3 Drawing a rectangle

Lets make another shape. The next simplest shape we can draw is a rectangle. Just like the create_line() method, we need to give the create_rectangle()two points; the top left point and the top right point. Also, we can optionally give the create_rectangle() method a “fill” paramater, which takes in a color, to fill the rectangle with.
